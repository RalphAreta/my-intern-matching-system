# Push to GitHub - Instructions

## Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `my-intern-matching-system`
3. Description: "Python desktop application for internship recommendations using collaborative filtering"
4. Choose **Public** or **Private**
5. **DO NOT** initialize with README, .gitignore, or license (we already have these)
6. Click **Create repository**

## Step 2: Push Your Code

After creating the repository, run:

```bash
git push -u origin main
```

Or if you need to update the remote URL:

```bash
git remote set-url origin https://github.com/YOUR_USERNAME/my-intern-matching-system.git
git push -u origin main
```

## Alternative: Use the provided script

Run `push_to_github.bat` after creating the repository on GitHub.
